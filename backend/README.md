# Backend

Commands:

- dev: `npm run dev` (requires Postgres running)
- build: `npm run build`
- start: `npm run start`
- test: `npm run test`

Environment variables: create a `.env` file in the project root with the following example values (update as needed):

```
DATABASE_URL=postgres://postgres:postgres@db:5432/app_db
PORT=4000
```
